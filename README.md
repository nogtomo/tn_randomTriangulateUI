# tn_randomTriangulateUI
制約付きランダム三角化ツール（Maya Python）
*************************************************
概要
tn_controlledTriangulateUI は、
モデルの外周や高さなどの見た目を維持したまま、内部構造だけをランダム化して三角化できる
Autodesk Maya 用の Python ツールです。

単純なランダム変形ではなく、「どこを固定し、どこをランダムにするか」を明確に指定できるため、
破綻しにくく、再現性のあるランダム化が可能です。
*************************************************
主な特徴
	選択したポリゴンメッシュに対して実行
	外周（境界）を固定し、シルエットを保持
	固定したい軸（X / Y / Z）をUIで選択可能
	指定した軸以外のみランダムに変形
	Seed指定による結果の再現性
	全フェースの三角化（Triangulate）
	Undo対応・複数オブジェクト対応
	UI操作で直感的に使用可能
*************************************************
想定される用途
	地形・床・壁など、フラットなメッシュのランダム化
	規則的すぎる三角割りを避けたい場合
	Houdini（RBD / Vellum / ノイズ処理）前の下処理
	授業・教材での「制約付きランダム」の説明
*************************************************
基本的な使い方

① モデルを選択
	ポリゴンメッシュを選択（オブジェクト／コンポーネントどちらでも可）
	Planeの場合は Subdivisionを増やすと効果が分かりやすい
② UIで設定
Vertex Randomize
	Jitter Amount　→ ランダム移動量（0.01〜0.03推奨）
	Seed　→ 同じ値で同じ結果を再現可能
	Lock Axis　→ チェックした軸は動かさない
例：Lock Y ON → 高さ固定
	Lock Boundary　→ 外周を固定し、全体の形状を保持
	Use Object Space　→ モデルが回転している場合に使用
Triangulate
	Triangulate　→ 全フェースを三角化
	Random Flip Edges（任意）　→ 三角割りの向きをさらにランダム化
	Keep History　→ ヒストリ保持の有無
③ 実行
Run on Selected をクリック
*************************************************
推奨設定例
	床・地形メッシュ（形状保持）
	Lock Boundary：ON
	Lock Y：ON
	Lock X / Z：OFF
	Jitter：0.02
	Triangulate：ON
*************************************************
注意点
	外周固定は「境界を持つメッシュ（Plane等）」を前提としています
※ 完全に閉じたモデル（球・箱など）では境界が存在しません
	Jitter値を大きくしすぎると、意図しない変形が起こる可能性があります
	表示確認は Wireframe on Shaded を推奨します
*************************************************
このツールの考え方
本ツールは
「ランダム＝適当」ではなく、
「制約を決めた上でのランダム」
というプロシージャル的思考を重視しています。
	何を守るか（外周・高さ・軸）
	どこを自由にするか（内部構造）
	再現できるか（Seed）
